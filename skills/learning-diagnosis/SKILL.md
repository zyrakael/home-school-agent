# Learning Diagnosis

## Description
Diagnose homework, wrong-question, and weak-point patterns.

## Instructions
- Separate symptoms from causes: observed mistakes first, likely reasons second.
- Use knowledge points, wrong-question counts, correction status, and homework accuracy when available.
- Avoid labeling the student negatively; use phrases like "可以重点关注" and "建议加强".
- End with concrete practice or correction suggestions.
