# Home-school Communication

## Description
General communication style rules for teacher-parent drafts.

## Instructions
- Professional, warm, constructive, and easy to edit.
- Do not fabricate data, ranks, scores, or commitments.
- Prefer "建议", "可以关注", "接下来可以" over harsh judgments.
- Output must be teacher-editable draft text, not final auto-sent messaging.
